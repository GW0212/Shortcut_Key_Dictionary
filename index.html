단축키 사전

Ctrl + C + V + Z + Space
🔍
All Programs

전체적으로 많이 쓰는 단축키

운영체제와 브라우저, 대부분의 프로그램에서 공통으로 동작하는 기본 단축키입니다.

Google Sheets

구글 스프레드시트 단축키

표 작업, 서식 지정, 시트 이동을 빠르게 처리할 수 있는 단축키 모음입니다.

Microsoft Excel

엑셀 단축키

데이터 입력, 서식 지정, 시트 탐색을 빠르게 만들어주는 단축키입니다.

Microsoft PowerPoint

파워포인트 단축키

슬라이드 작성, 발표 진행, 개체 정리에 유용한 단축키입니다.

Microsoft Word

워드 단축키

문서 서식 지정과 편집 속도를 높여주는 단축키입니다.

Adobe Photoshop

포토샵 단축키

도구 전환, 레이어 작업, 화면 확대·축소에 자주 쓰이는 단축키입니다.

Visual Studio Code

VS Code 단축키

코드 작성과 탐색, 디버깅 속도를 높여주는 단축키입니다.

Google Chrome

크롬 브라우저 단축키

탭 관리와 페이지 탐색을 더 빠르게 해주는 단축키입니다.

Notion

노션 단축키

페이지와 블록을 빠르게 만들고 정리할 수 있는 단축키입니다.

Slack

슬랙 단축키

채널 이동과 메시지 작성을 더 빠르게 해주는 단축키입니다.

Zoom

줌(Zoom) 단축키

회의 중 음소거, 화면 공유, 녹화 등을 빠르게 제어하는 단축키입니다.

Figma

피그마 단축키

도형·텍스트 도구 전환과 레이아웃 작업에 자주 쓰이는 단축키입니다.

Hancom 한글

한글(HWP) 단축키

문서 작성과 서식 지정에 자주 쓰이는 한글 프로그램 단축키입니다.
